Features and Tools

TEXT TO SPEECH TOOL

.

Engineered by R. Shailesh.

OneWorld’s latest Text-To-Speech tool allows users to enter text files and have them read aloud to them. This feature is geared towards visually impaired individuals and those who suffer from sight problems. Click on the image towards the right to interact with this feature.

VIDEO CAPTIONING TOOL

.

Engineered by N. Varma.

OneWorld offers a top-notch, high-quality video captioning tool. With this, users can enter a video file, and the program will return live captioning to go along with this video. This is especially useful for users with hearing difficulties and auditory impairments. To use this feature, click on the image towards the left of the screen.

AUDIO CAPTIONING TOOL

.

Engineered by N. Varma.

Similar to the video captioner, the OneWorld Audio Captioning feature allows for users to enter audio files. These can range from recorded messages, podcasts, audiobooks, etc. After that, the user should enter a text file that goes along with the inputted recording so that the software will have something to caption off of. This feature will mention which specific file types are allowed and prohibited within the webpage. Click on the image towards the right to interact with this feature.
